# PI-lot
### A Raspberry Pi Based Hand-Held Communication Device--
Created By: Lindsay Hall and Hannah Deyst

March - May 2024

The PI-lot is a Raspberry Pi based messenger with the ability to receive and reply to messages
through a flask application. This flask app is run on the PI-lot and accessed by a control computer that will be sending out the messages. Apart from written communications, key features include: a database to track and display messages previously sent to or received from the PI-lot, audio functionality for written and received messages, and a comprehensive user-guide. Physical components consist of a Raspberry Pi with a built-in audio jack, an external LCD display with five buttons, and a bluetooth keyboard.

This messenger is ideal for cross-workplace communication in situations where compact-size
and focused functionality is vital. Though just a prototype, it could easily be scaled to create a network of devices that receive messages from the control computer and would benefit from added encryption.
The PI-lot, if modified in a slightly different direction, could also be marketed as a toy for younger children who are not yet old enough to have a phone but want the ability to send messages to peers. Of course, durability would be a more important feature than encryption of messages in this case.

Ultimately, the PI-lot prototype is a first step towards a full office messaging system
or a children’s toy. Though vastly different directions, it can certainly be expanded to reach either demographic.
